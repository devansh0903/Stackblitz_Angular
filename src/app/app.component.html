Myth vs Fact
No Myth Facts
1 Only for creating new products Find innovative solutions for services,process etc
2 Increase risk of failure Actually weeds out failures more effectively

Features of Design Thinking

  • Finding
  • Having
  • Improving
  • Creating
  • Addressing

Steps of Design Thinking


1. Empathize

This is where you get information from domain expert and get to know more details about the project

2. Define

Reframe and create human centric problem statements

3. Ideate

Brainstorm Radical Ideas

4. Prototype

Create objects and experiences

5. Test

Testing along with customer to refine solution and gather data
